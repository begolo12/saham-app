## Endpoints Summary

### Authentication
- `POST /api/auth/login` — Login (rate limited 5/min/IP)
- `POST /api/auth/refresh` — Refresh access token
- `GET /api/auth/me` — Get current user

### Stocks
- `GET /api/stocks` — List top stocks (default 30)
- `GET /api/stocks/{symbol}` — Stock detail
- `GET /api/stocks/{symbol}/history` — Historical prices
- `POST /api/stocks/batch` — Multiple stocks at once
- `GET /api/stocks/search?q=` — Search by name/symbol

### Market
- `GET /api/market/summary` — IHSG + indices
- `GET /api/market/regime` — Market regime + macro

### Portfolio (auth)
- `GET /api/portfolio` — Positions + summary
- `POST /api/portfolio/positions` — Add position
- `DELETE /api/portfolio/positions/{id}` — Remove position
- `GET /api/portfolio/daily-report` — Daily report

### News
- `GET /api/news/{symbol}` — News for symbol (cached 30 min)
- `GET /api/news/sentiment/{symbol}` — Aggregate sentiment

### Learning (auth)
- `GET /api/learning/evaluations` — Historical accuracy
- `POST /api/learning/evaluate` — Trigger evaluation
- `GET /api/learning/recommendations` — Past recommendations

### Signal Accuracy (auth)
- `GET /api/accuracy` — Full accuracy dashboard
- `GET /api/accuracy/summary` — Compact version

### Admin (superadmin)
- `GET /api/admin/users` — List users
- `POST /api/admin/users` — Create user
- `DELETE /api/admin/users/{id}` — Delete user

### Health
- `GET /api/health` — Server health + versions
- `GET /` — HTML health check

## Authentication

Semua endpoint (kecuali `/api/auth/login`, `/api/auth/refresh`, dan `/api/health`) butuh Bearer token.

```
Authorization: Bearer <access_token>
```

Access token expire 15 menit. Pakai `/api/auth/refresh` untuk dapat token baru.

---

## Rate Limiting

- **Login**: 5 percobaan/menit/IP
- **General API**: 200 requests/menit/IP

Response kalau limit exceeded:
```json
HTTP 429 Too Many Requests
{"detail": "Too many requests. Please try again later."}
```

## Error Format

Standar FastAPI:
```json
HTTP 4xx/5xx
{"detail": "Error message"}
```

Validation errors (422):
```json
{
  "detail": [
    {
      "loc": ["body", "username"],
      "msg": "field required",
      "type": "value_error.missing"
    }
  ]
}
```
